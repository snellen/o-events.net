About this example

YAML support for rtl (right to left) languages is provided as an incremental add-on to the framework core files and all parts of the screen layout, see: Section 5.4:Add-ons.

Check out the XHTML source and the stylesheets of this example for more information about its usage.

Note: Within rtl (right to left) language content you can switch back to ltr (left to right) direction by using CSS-class .text-ltr, provided in base-rtl.css.

Dummy content for testing purposes can be generated with adhesiontext.
