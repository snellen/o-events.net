About this example

This layout example demonstrate the usage of Dirk Ginaders "Accessible Tabs" plugin (v1.5) and the SyncHeight plugin for jQuery.

These plugins are released under GPL and MIT License and therefore can be used standalone without violating YAML's license conditions.

For the latest versions of both plugins, visit Dirk Ginaders project homepage at Github. Fore more information, see Section 5.4: Add-ons.
