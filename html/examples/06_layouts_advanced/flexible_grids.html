About this example

This example demonstrates the usage of YAML's flexible grid elements (subtemplates) for layout purposes. There are no columns (#col1, #col2 or #col3) used in the markup. The grid elements can be nested and therefore provide an almost infinite variety of page designs.

This example is explained in detail in the documentation.
